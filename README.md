go-plinq
========

PLINQ library for go, support the lazy evaluated and can use channel as source


## Outstanding Linq operatoin:
* 过滤操作符：
Skip, SkipWhile, Take, TakeWhile

* 选择：
SeleteMany

* 量词/Quantifiers：
Contains
All, Any, AnyWith, 
SequenceEqual

* 生成器方法
Range， Repeat

* 元素运算符/Element Operators
ElementAt, First, FirstBy, Last, LastBy, Single,
